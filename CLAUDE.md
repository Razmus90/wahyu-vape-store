# Wahyu Vape Store

Next.js e-commerce app for vape products. Uses Supabase (Postgres) for data, Olsera API for product sync, Midtrans for payments.

## Agent skills

### Issue tracker

Issues live in GitHub Issues. See `docs/agents/issue-tracker.md`.

### Triage labels

Labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo. One `CONTEXT.md` + `docs/adr/` at root. See `docs/agents/domain.md`.
